Quick Start Guide

Step 1: Install Rust

$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Step 2: Install TexCreate

$ cargo install texcreate

Step 3: Create New TexCreate Project

$ texcreate init
Would you like to use the default config.toml? (y/n):
n
Initializing custom config.toml...
// Enter author name:

// Enter project title:

// Enter date:

// Enter project name:

//////////////////////////////////////

// Novel => A project to begin writing that novel you always delay
// Beamer => A project to start making presentations in LaTeX
// Lachaise => A project to write neat, nice documents/assignments
// Enter template:

// Enter font size:
11
// Enter document class:
article

$ cd Project_Name
$ ls
