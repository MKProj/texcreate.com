Quick Start Guide

Step 1: Install Rust

$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Step 2: Install TexCreate

$ cargo install texcreate

Step 3: Create New TexCreate Project

$ texcreate init
